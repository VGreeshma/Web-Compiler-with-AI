# 💻 Web Complier with AI(C/C++)

## 🌟 About the Project
Web Complier with AI is an AI-powered web compiler that allows users to write, run, debug, and download C/C++ programs directly in the browser. It provides a smooth coding experience using Monaco Editor and supports AI-based error assistance.

---

## 🚀 Features
- ✍️ Write C/C++ code using Monaco Editor (VS Code-like experience)
- ▶️ Run and compile code in real-time
- ❌ Error detection and display
- 🤖 AI-based debugging assistance
- 💾 Download code as file (like Notepad)
- 🌐 Multi-page UI with smooth navigation

## ⚙️ How to Run Locally

### 
1. Clone the repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

2. Run Backend
cd src/backend
pip install -r requirements.txt
python app.py

3. Run Frontend
npm install
npm start
